The netGALAXY Studios API plugin allows you to retrieve content in order to create a native app for iOS and/or Android using our app building platform

== Description ==

netGALAXY Studios API plugin is a fork of JSON API created by lcollett.

netGALAXY Studios API plugin creates a communication interface between your Wordpress website and your netGALAXY Studios account. 

It is used to retrieve content from your Wordpress website so that you can sync it with your native app created with netGALAXY Studios.

== Installation ==

1. Upload the gb-json-api folder to the /wp-content/plugins/ directory or install directly through the plugin installer.

1. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer
